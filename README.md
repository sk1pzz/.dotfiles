# .dotfiles

## Config

#### Keymaps

- `jk`: Exit insert mode
- `<leader>nh`: Clear search highlights
- `<leader>+`: Increment number
- `<leader>-`: Decrement number

Windows

- `<leader>sv`: Split window vertically
- `<leader>sh`: Split window horizontally
- `<leader>se`: Make splits equal size
- `<leader>sx`: Close current split

Tabs

- `<leader>to`: Open new tab
- `<leader>tx`: Close current tab
- `<leader>tn`: Go to next tab
- `<leader>tp`: Go to previous tab
- `<leader>tf`: Open current buffer in new tab

rmagatti/auto-session

- `<leader>wr`: Restore session for cwd
- `<leader>ws`: Save session for auto session root dir

numToStr/Comment.nvim

- `<leader>ggc`: Toggles the current line using linewise comment
- `<leader>gbc`: Toggles the current line using blockwise comment
- `<leader>[count]gcc`: Toggles the number of line given as a prefix-count using linewise
- `<leader>[count]gbc`: Toggles the number of line given as a prefix-count using blockwise

nvim-tree/nvim-tree.lua

- `<leader>ee`: Toggle file explorer
- `<leader>ef`: Toggle file explorer on current file
- `<leader>ec`: Collapse file explorer
- `<leader>er`: Refresh file explorer

  Explorer
  - `a`: Create file or directory
  - `c`: Copy
  - `d`: Delete
  - `D`: Trash
  - `e`: Rename
  - `E`: Expand all
  - `p`: Paste
  - `r`: Rename
  - `W`: Collapse all
  - `x`: Cut

kylechui/nvim-surround

- `ysiw)`: (surround_words)
- `ysiw(`: ( surround_words )
- `ys$"`: "make strings"
- `ds]`: delete around me!
- `dst`: remove HTML tags
- `cs'"`: "change quotes"

nvim-telescope/telescope.nvim

- `<leader>ff`: Fuzzy find files in cwd
- `<leader>fr`: Fuzzy find recent files
- `<leader>fs`: Find string in cwd
- `<leader>fc`: Find string under cursor in cwd
- `<leader>ft`: Find todos

folke/todo-comments.nvim

- `]t`: Next todo comment
- `[t`: Previous todo comment
